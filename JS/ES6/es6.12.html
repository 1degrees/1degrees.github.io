# Async + await

async函数是什么？一句话，async函数就是Generator函数的语法糖。主要用于异步操作处理。

function timeout(duration = 0) {
    return new Promise((resolve, reject) => {
        setTimeout(resolve, duration);
    })
}

var p = timeout(1000).then(() => {
    return timeout(2000);
}).then(() => {
    throw new Error("hmm");
}).catch(err => {
    return Promise.all([timeout(100), timeout(200)]);
})

WARNING

支持通过polyfill

为了支持Promise，你必须包括babel-polyfill。
