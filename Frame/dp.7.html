# 迭代器模式

迭代器模式是指提供一种方法顺序访问一个聚合对象中的各个元素，而又不需要暴露该对象 的内部表示。迭代器模式可以把迭代的过程从业务逻辑中分离出来，在使用迭代器模式之后，即 使不关心对象的内部构造，也可以按顺序访问其中的每个元素。

# 1. 迭代器模式应用

如 ES6-Iterators + For..Of 遍历数组

let arr = [1, 2, 3, 4]
for(let val of arr){
    console.log(val)
}

如 ES6-迭代器函数

function* foo(){ 
  yield 1; 
  yield 2; 
  yield 3; 
}; 

for(let o of foo()) { 
  console.log(o); 
}

自定义实现迭代器

function Iterator( obj ){ 
    let current = 0; 
    let next = function(){ 
        current += 1; 
    }; 
    let isDone = function(){ 
        return current >= obj.length; 
    }; 
    let value = function(){ 
        return obj[current]; 
    }; 
    return { 
        next: next, 
        isDone: isDone, 
        value: value 
    } 
};

var iterator1 = Iterator([ 1, 2, 3 ]); 
while(!iterator1.isDone()){
    console.log(iterator1.value())
    iterator1.next(); 
}

自定义实现对象类型迭代器

function Iterator( obj ){
    let current = 0; 
    let keys = Object.keys(obj);
    let next = function(){ 
        current += 1;
    }; 
    let isDone = function(){ 
        return current >= keys.length; 
    }; 
    let value = function(){ 
        return [keys[current], obj[keys[current]]]; 
    }; 
    return { 
        next: next, 
        isDone: isDone, 
        value: value 
    } 
};

var iterator1 = Iterator({a: 1, b: 2, c: 3 });
for(;!iterator1.isDone();iterator1.next()) {
    let [key, value]=iterator1.value()
    console.log(key, value)
}

# 小结

迭代器模式是一种相对简单的模式，简单到很多时候我们都不认为它是一种设计模式。目前 的绝大部分语言都内置了迭代器。
