# 面向对象设计特征

面向对象的三个基本特征是：封装、继承、多态。

# 1.封装

封装最好理解了。封装是面向对象的特征之一，是对象和类概念的主要特性。

封装，也就是把客观事物封装成抽象的类，并且类可以把自己的数据和方法只让可信的类或者对象操作，对不可信的进行信息隐藏。

# 2.继承

面向对象编程 (OOP) 语言的一个主要功能就是“继承”。继承是指这样一种能力：它可以使用现有类的所有功能，并在无需重新编写原来的类的情况下对这些功能进行扩展。

通过继承创建的新类称为“子类”或“派生类”。

被继承的类称为“基类”、“父类”或“超类”。

继承的过程，就是从一般到特殊的过程。

要实现继承，可以通过“继承”（Inheritance）和“混合”（minxs）来实现。

在某些 OOP 语言中，一个子类可以继承多个基类。但是一般情况下，一个子类只能有一个基类，要实现多重继承，可以通过多级继承来实现。

# 3.多态

多态性（polymorphisn）是允许你将父对象设置成为和一个或更多的他的子对象相等的技术，赋值之后，父对象就可以根据当前赋值给它的子对象的特性以不同的方式运作。简单的说，就是一句话：允许将子类类型的指针赋值给父类类型的指针。

实现多态，有二种方式，覆盖，重载。

覆盖，是指子类重新定义父类的虚函数的做法。

重载，是指允许存在多个同名函数，而这些函数的参数表不同（或许参数个数不同，或许参数类型不同，或许两者都不同），由于JavaScript语言的特性直接重载是无法实现

# 4.实战应用

如下案例：有页面A，页面B ，页面B中的包含页面A大部分特征。使用面向对象特性去编写代码如下，大家可以copy 以下代码到浏览器，仔细观察原型链；

class PageA {
    constructor(){
        this.name = '页面A选择框'
        this.age = '页面A创建时间'
    };

    addEvent(){
        console.log('A 页面事件监听', this)
    };

    init(){
        console.log('A 页面初始化')
    };

    destroy(){
        console.log('A 页面销毁')
    };
}

class PageB extends PageA {
    destroy(){
        console.log('利用多态性覆盖A页面destrory，生成B 页面销毁方法')
    };

    addEvent(){
        super.addEvent(); //复用父类方法，this指向子类本身
        console.log('利用多态性覆盖A页面addEvent，生成B 页面销毁方法')
    };

    addPage(){
        console.log('B 页面添加方法')
    };
}

var a = new PageA()
var b = new PageB()
console.log(a, b)
b.addEvent();
